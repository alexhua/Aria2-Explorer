How to Use

Get started with Aria2 Explorer in 4 simple steps

1

Install Aria2

For Windows users, install Aria2 Manager from Microsoft Store. Other platforms, download Aria2 utility.

2

Run Aria2

For Windows, run Aria2 Manager. Other platforms, run aria2c --enable-rpc in Terminal.

3

Install Extension

Install Aria2 Explorer from Chrome Web Store or Microsoft Edge Add-ons.

4

Configure & Enjoy

Enable auto-capture in extension options and configure as needed. Start downloading!
