How to Use

Get started with Aria2 Explorer in 4 simple steps

1

Install Aria2

For Windows users, install Aria2 Manager from Microsoft Store. Other platforms, download Aria2 utility.

2

Run Aria2

For Windows, run Aria2 Manager. Other platforms, run aria2c --enable-rpc in Terminal.

Install Aria2 Explorer from Chrome Web Store or Microsoft Edge Add-ons.

4

Configure & Enjoy

Enable auto-capture in extension options and configure as needed. Start downloading!

Frequently Asked Questions

What's the difference between Aria2 Explorer and Aria2 Manager?

Aria2 Explorer is a browser extension that captures downloads and sends them to Aria2. Aria2 Manager is a desktop application that runs the Aria2 engine with BT/torrent support and management UI.

Does Aria2 Manager support BT/torrent downloads?

Yes! Aria2 Manager has full BT/BitTorrent support including magnet links, .torrent files, DHT, peer exchange, UPnP port mapping, and automatic tracker updates.

Can I use them together?

Absolutely! They work perfectly together. Install Aria2 Manager for the download engine, then add Aria2 Explorer extension to capture browser downloads automatically.

Is it free and safe to use?

Yes, both Aria2 Explorer and Aria2 Manager are completely free and open-source. They're safe, lightweight, and don't contain ads or malware.
